# Learning Notes

This directory contains distilled notes from the original design corpus in `overnight-ai-aesthetic/`.

Workflow:

1. Read high-priority articles from `overnight-ai-aesthetic/aggregate/reading-priority-list.csv`.
2. Extract reusable design principles, cases, anti-patterns, and prompt/review rules.
3. Record processed files in `processed-manifest.csv`.
4. Keep original articles until deletion is explicitly approved.
