RGB edo HSL kolore-eredutan oinarritu daitezke. CSS kodean (eta antzina HTML) zenbakizko balio gisa zehazten dira, nahiz eta kolore batzuk ingelesezko izenekin ere erabil daitezkeen. Web orrien egileek hainbat modutan zehaztu ditzakete zenbakiz web dokumentu bat osatzen duten koloreak: RGB ereduko kode hamaseitarra erabiliz (adibidez, gorria: #FF0000) RGB ereduko balio hirukoitza erabiliz, 0tik 255era hamartarrean edo ehunekotan (adibidez, gorria: 250 0 0) HSL ereduko erreferentziak erabiliz (adibidez, gorria: 0° 100% 50%) Kodifikazioa sinplifikatzeko, egileek 216 koloreko paleta baten koloreak erabil ditzakete, ingelesezko izendapen batean oinarritutako hitz gako baten bidez (adibidez, gorria: Red). Kodeketa[aldatu | aldatu iturburu kodea] Pantailetan koloreak sortzeko, sintesi gehigarriko sistema erabiltzen da, zeinak hiru luminoforo mota erabiltzen dituen (gorria, berdea eta urdina,) bistaratze-distantzia normal batean fusionatzeko bezain txikiak. Zeharkatzen dituen korronte elektrikoaren intentsitateak zehazten du beraien argitasuna. Giza ikuspegiak, kolore bakoitzerako, ehun bat argitasun maila bereiz ditzake. Beraz, luminoforoen korronteak zazpi edo zortzi bitetan kuantifikatu daitezke. Hortaz, zortzi biteko kodifikazioa erabiltzen da, byte-a bezala prozesadoreen oinarrizko elementuari dagokiona. Kodeketa informatiko sinpleena pixel bakoitzerako, hau da, irudiaren edozein elementutarako, byte batean hiru balio ematean datza, argitasunari zuzenean dagozkionak. HTML eta CSS ereduek bi kodeketa onartzen dute: bai rgb(45,40,15), non zenbakiek gorriaren, berdearen eta urdinaren ehunekoa adierazten duten, bai #08A12C notazio kondentsatua, non # ondoren kodeak hiru balio hamaseitar dituen. HTML eta CSS ereduek deskribapen intuitiboagoa onartzen dute, HSL (Hue Saturation Lightness, Tonua Asetasuna Argitasuna), non tonua 0 (gorria) eta 360° arteko angelu batez definitzen den, berdea 120 eta urdina 240 izanik, eta asetasuna eta argitasuna ehuneko batez. HTML izenak[aldatu | aldatu iturburu kodea] HTML 4.01 bertsioaren zehaztapenak [1] 16 kolore definitzen ditu, ingelesezko 18 gako-hitzen bidez izendatuak (bi sinonimo pare daude), eta horien letra larri eta xeheek ez dute eraginik.
