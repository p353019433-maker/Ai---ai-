# 品牌与平面经典（循环6）

## Pentagram
- 1972 创立，五合伙人（Fletcher / Crosby / Forbes / Grange / Kurlansky）
- **架构**：扁平、合伙人制、无 CEO / CFO / 董事会
- "loose, horizontal organisation of equals" — 灵感来自 1956 "This Is Tomorrow" 展
- 24 个合伙人，**收入均分**，自筹自持
- 业主只能是设计师
- "Why give up control to a bunch of people who probably have no interest in design?"
- 案例：Citibank / Saks / United Airlines / Warner Bros / Daily Show 图形 / Pink Floyd 包装 / High Line / Public Theater

## Massimo Vignelli
- 哲学：**simplicity, clarity, timelessness**
- 目标："visually powerful, intellectually elegant, and above all timeless"
- "Design is one" 设计是一个整体
- "if you can design one thing, you can design everything"
- 客户：Knoll / Bloomingdale's / IBM / American Airlines
- 1972 纽约地铁图：清晰优先于地理准确

## Milton Glaser
- "reinvigorated interpretations of historical styles"
- Push Pin Studios 重塑设计师/插画家/视觉文化
- 风格：playful psychedelic + 控制色爆 + 剪影 + 粗几何
- **I♥NY 标志免费送纽约市**，"hoping it would become public property"
- 著 "Art is Work"：艺术即工作，拒绝艺术/日常的分离

## Stefan Sagmeister
- **Beauty 展（2019）**：通过设计/产品/建筑/城市规划探索"美"
- **Happy Show（2012）**：十年幸福研究，实验性 typography + 信息图 + 装置
- 探索 happiness 如何被理解与塑造

## April Greiman
- **"Does It Make Sense?"（1986, Design Quarterly #133）**：反抗现代主义的客观/理性/男性气质
- 自我描述："a magazine, which is an artwork, which is an object, which is... crazy"
- "transmedia" 跨媒体：type layering + 混合图像 + 数字技术
- 拒绝"graphic designer" 称号，自称"transmedia artist"
- 把欧洲新设计带到美国 + 加州后现代融合

## Wim Crouwel
- 网格 + 排版（International Typographic Style 传承）
- **New Alphabet (1967)**：专为 CRT 显示器，只用横竖笔画的字体
- "over-the-top and never meant to be really used"
- 把技术限制变为创作动力
- Stedelijk 博物馆视觉（1964-）

## Saul Bass
- 目标："reach for a simple, visual phrase that tells you what the picture is all about and evokes the essence of the story"
- 哲学："making the ordinary extraordinary"
- **Logo 平均寿命 34 年**：AT&T (1983)、United Airlines (1974)、Bell (1969)
- **电影海报变革**：象征化、简化、传达电影精髓
- **Title Sequences 革新**：kinetic typography + 动画剪纸
- Psycho / Vertigo / North by Northwest / Goodfellas

## Paula Scher
- 字体作为主要设计工具
- 风格：eclectic typography — 历史字体重生 + 当代图像
- **Public Theater 1994-**：街头 typography + 涂鸦式并置 + 平面色
- Pentagram NY 1991-（合作 MoMA、大都会、Microsoft）
- 教学 SVA / Cooper Union / Yale / Tyler

## Tibor Kalman
- 设计应有益于日常生活 + 提高社会意识
- 伦理设计 = 伦理客户
- 故意用 vernacular 民间风格抵抗主流 International Style
- Colors 杂志：专注"the rest of the world"，推动多元文化
- 挑衅式大师

## Herb Lubalin
- "the look and sound of words" — 痴迷字的形与声
- 字体选择改变解读
- 与 Ralph Ginzburg：*Eros* / *Fact* / *Avant Garde*
- *Avant Garde*：紧密贴合的字母组合 = "futuristic, instantly recognizable identity"
- 理想："I'm my own client. Nobody tells me what to do."

## 对 AI 去 AI 化的启示
1. **Vignelli "Design is one"**：跨媒介一致原则，AI 输出跨媒介常不一致
2. **Bass "essence + extraordinary"**：logo 要传达故事精华 + 让日常不平凡
3. **Pentagram "horizontal + quality"**：设计师是 owner，不是 tool
4. **Glaser "Art is Work"**：好设计 = 服务大众，不是炫技
5. **Kalman 民间 + 挑衅**：AI 喜欢"光滑", 真正有性格的反而是 vernacular
6. **Crouwel 限制为创作**：给 AI 限制（如"只能用 4 种几何"）反而能产好设计
7. **Scher 街头 typography**：让 typography 表达地点/文化，不要千篇一律
8. **Lubalin 字即形象**：AI 容易把字当信息载体，忽略字本身的视觉力量
9. **April Greiman 跨媒体**：不要把"网页"和"PPT" 当两套设计，要统一
10. **Sagmeister 美与幸福**：AI 默认"功能优先"，忽略美
