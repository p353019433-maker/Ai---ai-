## 6. 7 条具体原则（可抄）
1. **Research workshop positioning**。Nouvelle étiquette 把自己框为"studio and research workshop" = 比"工作室"高半档
2. **Multi-genre typefaces in one file**。Dinamo Arizona 把 Sans/Serif/Text/Mix/Flare 合到 1 file = 单一文件即系统
3. **Curated-zine homepage**。Dinamo 主页混 specimens + 硬件 + 出版 + NTS Radio show (foNTS) + Guest Essays = 主页即策展
4. **Locally customisable filters**。Rosetta Hyperglot 让用户发现**具体字体支持的语言** = 把"我有什么"变成"你用什么"
5. **"Suggestively incomplete" systems**。Nouvelle étiquette 的 ASCII identity 是"未完成"的 = 留下用户参与空间
6. **Sustainability as design problem**。Braungart/McDonough 命题 = "pollution is a design problem" = 重定义 design brief
7. **Cross-discipline jam-session collaboration**。Polestar × Grcic 复刻 Re:Move = 工业+设计+可持续的"jam session"

## Why it matters
French / type 研究的两个核心：**(a) studio = research workshop 的定位**，**(b) cross-discipline 协作的 jam session 模式**。对中小型工作室：把自己框高半档（research 不是说辞是身份）；对大企业：让 type 团队、产品团队、可持续团队在同一项目里"jam"。
