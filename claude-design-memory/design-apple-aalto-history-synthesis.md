## ⚠️ 诚实声明（来自子代理）
**Apple 集群里没有 Jony Ive 第一人称材料**——只有 Apple 2025-26 自己的 self-documentation。Snow White stripes / Bondi Blue iMac / unibody MacBook / "obsess over the seams" **都不在 corpus 内**。
**平面设计史不是** Bauhaus / Swiss / Müller-Brockmann / Helvetica 真正 corpus——唯一 Müller-Brockmann 是 norm 文章一句话；Helvetica 只是 Klim + Dinamo 的**反衬**。**Aalto 是唯一有深度的服务对象**。

## 2. Aalto (1898-1976) —— 真有深度
- 弧：**Nordic Classicism (1920s) → International Style (1930s) → organic modernism (后期)**
- **Paimio Chair (1932)** —— "Though buoyant as a spring cushion, the seat back is virtually unbreakable"
- **Savoy Vase (1936)** —— 灵感来源 "Eskimo woman's leather breech"，与 Hans Arp（达达）有关
- **Aalto Vase** (1936, Iittala) = 一体成型的 plywood bend 工艺
- **Viipuri Library (1927-35)** —— 芬兰/俄罗斯
- **Plywood 工艺历史**：从胶合板到弯曲胶合板
- **Muuratsalo** 实验屋：brick 试验
- 直接引语：见上 Paimio + Savoy

## 3. 平面设计史 —— 比预想稀薄
- **Bauhaus**：仅 HIT 的 ifa touring exhibition identity 一篇
- **Müller-Brockmann**：只有 norm 文章里**一句引语**
- **Helvetica**：Klim + Dinamo 的**反面教材**
- **真正有的**：Jan Lenica / Chip Kidd / Visual hierarchy (Gestalt + Von Restorff) / Saul Bass / Emily Oberman / Lucida Grande / Hershey fonts / Michael Bierut / Martí Guixé / Müller+Hess / Print magazine / Henry Wolf / Andrzej Pągowski / Studio w+e / ColorADD / Postage stamp design / Christoph Knoth / Saja Typeworks

## 4. 8 条具体原则（可抄）
1. **Resilience through geometry**（Aalto Paimio Chair "virtually unbreakable"）
2. **Humanist deviation from functionalism**（Aalto 在 Nordic Classicism → organic 后转向）
3. **Gesamtkunstwerk**（Vitra 引用 Panton Visiona II）= "total work of art"
4. **System-with-intention**（Apple WWDC25 2025 新词）
5. **Variable optical sizing as constitution**（Apple SF Pro 在 size-specific 变体）
6. **Helvetica as logistics typography**（Klim + Dinamo 的反面用例）= 字体是 logistics 不 expression
7. **Inheritance through subversion**（Memphis 1981 反功能主义 = Sottsass "sensual and exciting"）
8. **Anti-signature as signature**（Müller-Brockmann grid 系统的"无我"）

## 5. 重要补充
- **Lucida Grande** (1999-2014 macOS) → SF Pro 取代
- **Hershey fonts** (1967, vector stroke fonts for early CRT) = 屏幕字体起点
- **Saul Bass** = 经典电影标题+logo+海报
- **Henry Wolf** = 1960s 摄影+设计
- **Andrzej Pągowski** = 波兰电影海报
- **Chip Kidd** = 1990s 至今 Knopf 书籍封面
- **Martí Guixé** = 西班牙 conceptual food design

## Why it matters
- Apple 2025-26 设计语言：Liquid Glass（玻璃质感 + concentricity）
- Aalto：humanist modernism 的**有机版本**（与 Bauhaus 严酷形成对比）
- 平面史真正的 21 世纪转向：foundries（Rosetta/Klim/Dinamo）+ 字体系统思维（参 [design-typefaces-2026-synthesis.md]）
