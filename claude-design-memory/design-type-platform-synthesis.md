## 6 条具体洞察
1. **Solarized 哲学 = 减法**——8+8 极简，**不是更多色而是更少色** = 跟 Pantone (几百色) 反着走
2. **Areal = "Frankenstein" 修复**——Arial 是"各种历史字体拼接"——Areal 把它当 system 重新做
3. **JAF size-specific = 500 年金属活字传统回归**——digital 时代"全尺寸一样" = 退化，**size-specific 优化**才是 500 年传统
4. **MDes Basel = Thesis-as-Exhibition** —— 设计硕士毕业 = 展览不是论文
5. **IPA 字体 = 设计给看不见的人**——语音学字体服务语言学家/口译员，是字体应用边缘领域
6. **Win32 7 元素 + removal-first** = 微软桌面设计系统的**7 元基本元素** + **减法优先**决策树

## Why it matters
- **Solarized 的 8+8 哲学** 对小/中产品 = 调色板"少就是多"，**对比度优先** = 给 a11y 加分
- **Areal 修 Arial** 对字体厂商 = 经典字号可以**重做而不照抄**
- **JAF size-specific** 对 UI 字体 = 不同 size 要有不同字形优化，**不是"全尺寸一样"**
- **MDes Basel** 对设计教育 = 毕业 = 展览，**不是论文**
- **IPA 字体**对边缘受众 = 设计应用不限于商业
- **Win32 removal-first** = **先想能去掉什么**，不是加什么
