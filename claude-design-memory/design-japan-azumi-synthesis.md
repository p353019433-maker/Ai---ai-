## 1. Azumi 夫妇
- **Shin** (Kobe, 1965) + **Tomoko** (Hiroshima, 1966)
- Kyoto City University of Art 相识 → 伦敦 RCA → **1995 共创 Azumi**
- **"Changing people's behaviour in a subtle way"** = 哲学题
- 1996 Salt & Pepper Shakers：**tilt-don't-shake** = 重新设计"用"的动作（用户行为改变）
- 1995 **Table = Chest** = dual function
- Edinburgh Fringe *Breath* 作品 = 身体+运动教育
- 超市 basket / 3D-wireframe 材质运用

## 2. Japanese Style 设计美学（Sunamita Lim 2007）
- 词典式来源：Google Books preview
- 关键术语索引：**wabi-sabi / shibui / tatami / shoji / fusuma / tokonoma / tsukabai / tansu / chanoyu / teahouse**
- 关键人物：Yanagi（民艺）/ Leach / Nakashima / **Ando**（建筑）/ Le Corbusier / Wright
- Lim 框定："doing more with less"

## 3. "Typically Japanese" 标签的反感
- Azumis "resented and rejected" the "typically Japanese" label
- **Nakamura**："craftsmanship is a spirit"（不是技法）
- **Shiratori** 反思："questioning need for more things"
- Industrial Facility **Chantry millimetric tweaks** = 西方设计师在日本语境里的小幅调整

## 4. Japan vs Western 设计（按源材料）
- Lim 的 "doing more with less"
- Azumis 反感 "typically Japanese" 标签
- Nakamura "craftsmanship is a spirit" 反转
- **"small alterations" vs Bauhaus/IDEO systems-talk** = 日本设计是微调不是体系
- 西方机构锚：Antonelli/MoMA / Design Museum / Landor / Wright

## 5. 4 条可抄决策
1. **Salt-shaker 孔位** = 重新设计"用"的动作 = **设计行为不是设计物件**
2. **Dual-function Table = Chest** = 形式压缩即功能
3. **Craft as spirit**（Nakamura）= 工艺不是技术
4. **Chantry millimetric tweaks**（Industrial Facility）= 小幅调整 = 精微

## Why it matters
日本设计不是"风格"——是 **"重新设计用户行为"**（Azumi 盐瓶）和 **"craft 是 spirit 不是 technique"**（Nakamura）。对西方设计师：学的是"少改动多观察"——日本设计成功不靠新对象，靠新用法。
