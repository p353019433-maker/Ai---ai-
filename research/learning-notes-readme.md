# Learning Notes

This directory contains distilled notes from the original design corpus in `corpus/research archive (old path)/`.

Workflow:

1. Read high-priority articles from `corpus-metadata/reading-priority-list.csv`.
2. Extract reusable design principles, cases, anti-patterns, and prompt/review rules.
3. Record processed files in `processed-manifest.csv`.
4. Keep original articles until deletion is explicitly approved.
