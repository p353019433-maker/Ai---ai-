# Knowledge — 设计知识库

这是一套整理后的设计知识库（原 `claude-design-memory`），不是原始爬取语料。目标是让 agent 在做 UI、网页、产品、PPT、视觉系统、AI 设计提示词时，有可引用的判断框架，而不是只凭"好看/不好看"。

> 原始语料在仓库另一侧的 [`../corpus/`](../corpus/)。本目录是消化后的成品。

## 最重要的入口

- [`MEMORY.md`](MEMORY.md)：完整索引（62 份文档，分 10 个主题目录）。
- [`00-handbooks/design-decision-handbook.md`](00-handbooks/design-decision-handbook.md)：按具体设计问题路由到对应文件。
- [`00-handbooks/design-ultimate-handbook.md`](00-handbooks/design-ultimate-handbook.md)：跨主题总纲。
- [`00-handbooks/design-philosophy-master.md`](00-handbooks/design-philosophy-master.md)：去 AI 化设计哲学总纲。
- [`09-meta/HANDOVER.md`](09-meta/HANDOVER.md)：接手说明。

## 核心判断

这套知识库反复围绕一个主题展开：

> AI 感来自平均、安全、对称、装饰代替结构；真实设计来自减法、层级、约束、材料诚实、具体传统和明确主见。
