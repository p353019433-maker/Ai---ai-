# 设计史关键时刻（七个运动）

## 一句话总结
设计史不是风格谱系，是一部"决策史"——每个运动都把上一代做错的事（装饰、垄断、华丽）反过来做了一遍，而当下 AI 时代正在重复 1990 年代 Swiss Style 统一感带来的同样错误。

## 1. 包豪斯 Bauhaus（1919-1933）

**时间地点**：1919 Walter Gropius 在魏玛创立，1925 迁德绍，1933 被纳粹关闭，共 14 年。Herbert Bayer、László Moholy-Nagy、Wassily Kandinsky、Paul Klee 任教。

**关键人物 + 代表作**：Herbert Bayer 设计 **Universal** 字体（1925-1930），几何无衬线、全小写、消除大小写区别。Bayer 后任 CCA 总裁主导 30,000 件企业收藏。

**核心决策**：
- 取消艺术家与工匠的阶级差别（"new guild of craftsmen"）
- 形式追随功能，几何 + 三原色 + 无装饰
- Bayer 主张**废除大写字母**——"更高效、更民主"
- Moholy-Nagy 用摄影蒙太奇取代插画

**当时争议**：保守派贴"非德意志""颓废艺术""共产主义智库"。包豪斯三迁校址，1933 关停。

**对今天的影响**：网格系统、几何无衬线（Inter / SF Pro）、原色配色、Apple 极简、Google Material Design 的源头。Vercel / Linear / Stripe 的极简感 = 包豪斯 + Rams 的 80 年后回声。

## 2. 瑞士派 Swiss Style（1950-1970）

**关键人物 + 代表作**：
- **Max Miedinger + Eduard Hoffmann 1957 设计 Neue Haas Grotesk**，1960 改名 **Helvetica**
- **Josef Müller-Brockmann** 为苏黎世 Tonhalle 音乐厅做的 **Musica Viva 海报系列**（1950s），用几何形状翻译音乐数学结构
- Müller-Brockmann 1981 写《Grid Systems in Graphic Design》= 网格圣经

**核心决策**：
- 网格（grid）作"理性骨架"，**Müller-Brockmann 自言"网格是辅助，不是保证"**
- 无衬线 + 客观 + 摄影代替插画
- Helvetica 旨在与 Akzidenz-Grotesk 竞争，提供"中立"瑞士现代感
- 不对称、左对齐、右边参差

**当时争议**：被批"冷""无灵魂""理性主义"。Wim Crouwel 评"应该中性，本身不应该有含义"。Erik Spiekermann 与 Martin Majoor 质疑过度使用，Majoor 评"rather cheap"。

**对今天的影响**：IBM / Lufthansa / Olivetti / 纽约地铁 / Apple / 加拿大政府 / BMW 都用 Helvetica。Bootstrap / ZURB Foundation / CSS Grid = Müller-Brockmann 网格的代码化。Stripe Docs / Vercel Dashboard = Swiss Style 数字化延伸。

## 3. 新浪潮 New Wave（1970s 末-1980s）

**关键人物 + 代表作**：
- **April Greiman 1986** 的《Design Quarterly #133》——三英尺乘六英尺折叠海报，她自己裸体覆盖符号与文字
- Weingart 在巴塞尔设计学院发起，传到 80s 美国西海岸
- **Zuzana Licko + Rudy VanderLans** 办 Emigre 杂志，做 Citizen Kane / Emperor / Mrs Eaves

**核心决策**：
- 打破网格——不一致字距、单词内多变字重、非 90 度排版
- 拥抱"丑"——破损照片、抖动字体、互相冲突的字体混排
- 拒绝"客观"——设计是表现，不是传达
- 把 Macintosh 视为创作工具而非排版工具

**当时争议**：《洛杉矶时报》批评 Greiman "制造混合图像的实验"——质疑"设计师"是否有权做这种东西。Helvetica 阵营纯粹派视她为异端。

**对今天的影响**：brutalist web / 野兽派网页 / 设计师个人主页反精致排版 / Webflow 模板的"非系统感"——全是 New Wave 21 世纪回声。Greiman 被 Steven Heller 称"现代与后现代、模拟与数字之间的桥"。

## 4. 孟菲斯 Memphis Group（1981-1988）

**关键人物 + 代表作**：
- **Ettore Sottsass** 1980.12.6 在米兰公寓召集成立，1987 解散
- 核心成员：Nathalie Du Pasquier / George Sowden / Michele De Lucchi / Matteo Thun
- Sottsass **Carlton 书架**（1981）——塑料层压板 + 几何碎块拼贴
- 其他：Ultrafragola 镜子（1970）/ Superbox 柜（1966）/ Valentine 打字机（1969）
- Karl Lagerfeld 与 David Bowie 是大藏家

**核心决策**：
- 反对现代主义"反功能"——故意做"反理性"
- 用色像 Pop Art + 1950 年代媚俗（kitsch）+ 印度马杜赖佛教寺庙纹样
- **拒绝"好品味"**——Sottsass 自言要"激进、有趣、出格"
- 拼贴、塑料层压板、几何 + 曲线 + 斜杠混搭

**当时争议**：被《纽约时报》式主流称"怪异""被厌恶""非设计"。功能主义阵营说"背叛了包豪斯"。

**对今天的影响**：2024-2026 Memphis 复兴、weird-core、Loewe 2024 春夏、Yinka Ilori 与 Morag Myerscough 公共装置、Figma 官网页面"克制的彩色"——全从 1981 那个米兰公寓来。Vaporwave 美学（2010s）也直接继承。

## 5. 极简主义 Minimalism（1990-2000）

**关键人物 + 代表作**：
- **Dieter Rams 1955-1997** 在德国 Braun 主导工业设计，**Rams 十诫**（1970s 成型）
- 代表作 Braun SK4 收音机（1956，外号"白雪公主的棺材"）/ T3 收音机 / Vitsœ 606 通用置物架（1960）
- **Jonathan Ive 1992-2019** 在 Apple 把同一哲学放大：iMac G3（1998）/ iPod（2001）/ iPhone（2007）/ iPad（2010）/ Apple Watch（2015）
- 2010 Ive 给 Rams 寄了一台 iPhone 附信："感谢你的工作给我的灵感"

**Rams 十诫**：
1. 创新
2. 有用
3. 审美
4. 易理解
5. **谦逊**——"像工具，留白给用户自我表达"
6. **诚实**——"不把产品伪装得更创新或更贵"
7. 持久——"不追赶时髦，所以永不过时"
8. 细节彻底
9. 环保
10. 尽可能少的设计——"Weniger, aber besser"（少而更好）

**当时争议**："冷""无灵魂""享乐缺失的设计"（design by anhedonia）。Bruce Mau 与后现代派对极简强烈反弹。批评者指出，**极简也成了一种"风格时尚"——与 Rams 初衷"不追逐时髦"自相矛盾**。

**对今天的影响**：Notion / Linear / Vercel / Arc 浏览器 / Stripe 文档 / Braun 复刻系列 / Rams 2025 获 World Design Medal——**2026 硅谷 UI 几乎是 Rams 十诫的字面执行**。

## 6. 后数字 / 数字原住民 Post-Digital（2010s+）

**关键人物 + 代表作**：
- **Bret Victor**（Dynamicland, Oakland）三场改变行业的演讲：
  - 《Inventing on Principle》（2012）
  - 《Stop Drawing Dead Fish》（2012）
  - 《Learnable Programming》（2012）
- Alan Kay 称他"当今世界上最伟大的 UI 设计头脑之一"
- Apple（2007-2011）参与 iPad 与 Apple Watch 早期概念
- **Frank Chimero**《The Shape of Design》（2012）+《The Web's Grain》（2015）——"做对自己有意义的项目"

**核心决策**：
- **"看见过程"（seeing the process）**——动态表征比静态截图更接近真相
- "为思考不可想之物而生的媒体"——工具决定能想出什么
- 反对"用计算机当快速纸模拟器"——计算机的真正属性是动态
- **变量字体**（OpenType 1.8，2016 由 Adobe/Apple/Google/Microsoft 联合发布）让一份字体文件含无限字重/字宽/光学尺寸

**当时争议**：被学院派批评"不切实际"——Victor 的 Dynamicland 至今未商业化。Chimero 批评 SaaS 化的"设计剧院"（design theater）。

**对今天的影响**：Figma / Replit / tldraw / Observable / Craft / v0.dev / Cursor——所有"实时可编辑、可看见过程"工具都站 Victor 肩上。变量字体让 2026 网页排版能随视口连续变形。

## 7. AI 时代（2020s+）

**时间地点**：2022-2026，全球。核心是生成式图像（Midjourney / Stable Diffusion / GPT-4o image）、生成式 UI（v0 / Lovable / Bolt）、生成式排版与品牌系统。

**关键人物 + 代表作**：暂无单一"奠基人"，但 **Frank Chimero 2025《Beyond the Machine》**是首批严肃论辩——把 AI 当"乐器"而非"工人"。

**核心决策**：
- 用提示词（prompt）代替工艺（craft）
- 接受"足够好"的中间值（mean）
- 把人降级为"修改者"而非"创造者"

**当时争议**：2026 设计界最大的内部争议。一派认为 AI 释放生产力；另一派（也是本节论点）认为它制造了 **"AI 出活感"**——表面无错、内里无我、缺乏手艺痕迹的同质化输出。

**对今天的影响**：所有 2026 想做"看起来不像 AI 做的"设计项目（anti-slop 风格 / Loewe 风格手工字体回归 / Tobias Beving 复古手绘海报）——**都是 1980s New Wave 反对 Swiss 统一感、Memphis 反对包豪斯理性主义的 21 世纪重演**。

## 7 条跨运动元原则

1. **每个运动都是上一代的反命题**：包豪斯反维多利亚装饰 → 瑞士派反包豪斯教条 → New Wave 反瑞士派冷感 → Memphis 反现代主义普遍真理 → 极简反 Memphis 装饰 → 后数字反极简无灵魂 → AI 时代反一切人造意义

2. **"少而更好"是普世追求，但"少"从不静止**：包豪斯的"少"是 1919 年的少，Rams 的"少"是 1970 年的少，Apple 的"少"是 2007 年的少

3. **网格是辅助不是保证**——Müller-Brockmann 1981 明说，但 90% 设计师忘了

4. **形式追随功能之后，永远会有一代说"功能追随形式"**——Sottsass / Greiman / Victor 都在不同时点重复

5. **设计争议 = 阶层争议**：包豪斯"非德意志"、Memphis"被厌恶"、AI"反人类"——争议背后是身份与权力

6. **工艺痕迹（痕迹、噪声、不完美）是反均值化的武器**：Universal 字母几何不完美、Greiman 图像噪点、Memphis 塑料层压板接缝、AI 时代手绘回归——都指向同一件事

7. **设计史不是进步史，是钟摆史**：每个十年都把上代做过头的地方反过来做

## 5 条反面教材（运动中走过头的地方）

1. **包豪斯把"无装饰"做成新教条**——纳粹指控"颓废"部分正确：它杀死了手工艺的偶然性
2. **瑞士派的"客观"是神话**——所谓"中立"实为特定欧洲理性主义男性审美，Crouwel 那句"中性"本身就不是中性的
3. **Memphis 把"反功能"做成新功能**——Carlton 书架几乎不能放书，沦为符号
4. **Rams 极简被硅谷学成"性冷淡"**——Rams 初衷是"谦逊给用户留白"，实际变"按钮不能多于 3 个"教条
5. **New Wave 的"打破网格"在 90s 变不会对齐**——Web 1.0 大量丑陋站点是该运动过度延伸

## 6 条与"AI 去 AI 化"的关系

1. **Bayer 的"全小写"决定**——废除大小写等级、追求视觉平等，是反 AI"层级模板"的最早宣言
2. **Müller-Brockmann 的"网格是辅助"**——AI 工具倾向"用模板代替判断"，网格哲学反过来说明"判断永远在网格之外"
3. **Greiman 的"个人 Mac"**——把工具掌握在自己手里，而不是被工具默认值吞掉，是 2026 anti-slop 设计师最该学的姿态
4. **Sottsass 的"反好品味"**——AI 默认输出"好品味"，Sottsass 1981 证明"坏品味"才有灵魂
5. **Rams 第 6 条"诚实"**——"不把产品伪装得更创新或更贵"——对 AI 给"塑料花朵戴真花瓣滤镜"最直接一击
6. **Victor 的"看见过程"**——AI 把"过程"藏进黑箱；要反 AI 出活感，最直接方法是**让"过程可见"——手稿、未完成、迭代痕迹全部展示**

**最终论断**：所有 2026 "反 AI 出活感"的设计决策，**几乎都能在设计史里找到原型**。包豪斯反装饰 / 孟菲斯反理性 / Greiman 反客观——历史已经替我们写好剧本，我们只需要重新排演。
