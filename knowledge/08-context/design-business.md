# 设计商业化（从知识到生意的八条路径）

## 一句话总结
设计商业化的本质不是"卖好看的图"，而是把"判断力 + 流程 + 系统 + 注意力"打包成可定价的产品——**可重复卖的，永远比只卖一次的贵一个数量级**。

## 一、设计咨询三种活法

**Pentagram**（1972 创立）：**全球最特殊合伙制设计公司**——23-24 名合伙人即股东、也是主创、也是客户第一对接人。"the only major design studio" 采用这种结构：没有 CEO、没有销售部、没有 PM 层级——**客户直接付钱给设计师本人**。

合伙人"collaboratively or independently, in friendship" 工作；离职/退休的合伙人成为 Partners Emeriti；新引入的可以是 Partner at Large。**关键商业逻辑**：合伙人退出机制是"买断股份"而非股权稀释，所以 Pentagram 可以无限保持 20+ 人的规模而不变成企业。

代表项目：Mastercard 2016 品牌升级（Michael Bierut + Luke Hayman）/ United Airlines 重塑 / MIT Media Lab 视觉系统 / Saks Fifth Avenue / Public Theater / SNL 48 季 logo / Warner Bros. 2019 重塑 / Daily Show 2005 演播室 / Harley-Davidson Museum 建筑 / Alexander McQueen 门店。

**Lippincott**（1943 创立）：走另一条路——9 个全球办公室、6 大服务（品牌建设 / 增长解锁 / 体验重塑 / 营销策略 / 客户互动 / 员工激励）、客户全是 Danaher / Nokia / Korean Air / LendingTree 这类大企业。

**Lippincott 不卖"logo"，卖"measurable growth"**——价格与商业结果挂钩，这正是 value-based pricing 的典型形态。代表项目：Samsung 1993 品牌重塑 / Walmart "Spark" logo / Starbucks 40 周年重塑 / Korean Air 2025 全球重塑。

**MetaDesign**（Erik Spiekermann 1979 柏林创立）：200 人规模、全球办公室，但股权几经易手（Lost Boys 收购 Spiekermann 离场、2001 后 Spiekermann 另起 United Designers Network 2007 更名 Spiekermann Associates）。**核心教训**：设计公司的人力资本就是公司本身，合伙人制度若不锁住，资产会被并购方抽走。

代表项目：BVG 柏林公共交通系统（1980s）/ Volkswagen / Audi / Düsseldorf Airport / Deutsche Bahn / Bristol Legible City 寻路系统。

**给独立设计师的启示**：不必模仿 Pentagram，但可以学它的"客户直接对接主创"——**这能让你保留 60-80% 原本被中间层抽走的利润**。

## 二、设计系统市场化的 3 种路径

1. **内部系统 + 雇主品牌**：Atlassian / Shopify Polaris / IBM Carbon 开源不是为了直接卖钱，而是让客户、合作伙伴、招聘候选人看到"我们做事很工程化"，降低 BD 与雇人成本
2. **Figma Community 模式**：Airtable / Atlassian / GitHub / Salesforce / Spotify 都在 Figma Community 免费公开自己的设计系统，Figma 借此成为设计协作事实标准
3. **开源框架 + 商业模板**：Tailwind CSS（MIT + Tailwind UI 付费）/ shadcn/ui（不发布 npm 包，只发布可复制粘贴的源文件）——**shadcn 绕过了"开源=被白嫖"的传统困境**

**设计 token 本身很难直接卖**（太轻、复制成本为零），但**围绕 token 的治理、迁移、审计、定制**是真实的高客单价服务。

## 三、DesignOps：把"设计混乱"变"可计费产品"

Superside 研究：大厂 DesignOps 团队通常包含 Design Program Manager / Design Recruiter / Design Tooling Lead / Design Researcher Ops / Design Systems Lead / Brand Ops。

**ROI 算式**：每个设计师每周节省 5 小时琐碎协调 × 100 设计师 × 52 周 = 26,000 工时 × $150/小时内部计费 = **$3.9M/年可量化收益**。

**对独立设计师来说**：把自己包装成"DesignOps 顾问"比"UI 设计师"贵 3-10 倍。可以从 "$15K/月 DesignOps 嵌入" 开始，把小客户培养成 $100K/年 retainer。

## 四、设计师个人品牌金字塔

1. **Twitter/X 长文** → 0 元 / 关注者
2. **Newsletter**（Substack / Buttondown）→ $5-10/月订阅或免费+赞助
3. **写书 / 出独立出版物**（Chimero 模式，144 页《The Shape of Design》）→ $20-40/本
4. **教课**（Domestika / Skillshare / 自有课）
5. **高客单咨询 / Coaching** → $5K-50K/项目

**Frank Chimero**："小而美"典范——纽约一人工作室 + 一本书 + 长博客。**他没有大型课程、没有 SaaS、没有团队**——靠高质量输出累积声誉。

**Robin Rendle**："大厂 + 副业"混合形态——Apple + Buttondown newsletter + 自由职业。**副业不超过 20% 精力**，但副业是个人品牌的复利。

**反直觉**：大多数设计师死在第 2 步（写了 newsletter 但没有分发），极少数能跳过第 3 步直接进第 5 步（靠声誉直接接咨询）。

## 五、项目定价的三角悖论 + 合同 4 必写

**三种定价法**：
- **时薪**（Time & Materials）$75-300/小时，天花板最低
- **固定价**（Fixed）报价前置、范围清晰、利润高但风险大
- **价值定价**（Value-based）按对客户业务影响收费，可达项目价的 5-20 倍

**真正的工作室会组合用**：
- Discovery / 战略阶段 → 固定价（$10K-50K）
- 执行 / 制作阶段 → 时薪或人月（业内常用 $15K-40K/月）
- 长期合作 → Retainer（$5K-30K/月）

**合同里必须写死 4 件事**：
1. **scope**（范围）
2. **deliverable**（可交付物）
3. **revision rounds**（修改轮数，通常 2-3 轮）
4. **kill fee**（项目中途终止费，通常 30-50%）

少写一条就会出现"客户让我改 17 轮然后说没想到这么贵"的惨剧。

**Stripe Adaptive Pricing 数据**：对订阅产品做基于地理位置的智能定价，能在 150+ 国家带来 4.7% 转化提升、5.4% LTV 提升。

## 六、设计奖项与曝光

| 奖项 | 模式 | 关键事实 |
|---|---|---|
| **Webby Awards**（1996 起） | 单条投稿 $495 | IADAS 3,000+ 评委 + People's Voice 公众投票；五词获奖感言已成文化符号 |
| **Red Dot**（1954 起） | 明确 pay-to-play | 获奖后 €1,950-7,500 费用；**function / aesthetics / usability / responsibility** 四项判据 |
| **iF Design Award**（1953 起） | 不强调"获奖后费用" | 6 大领域 70 类别，更像"声望标签" |
| **Awwwards** | SOTD + Developer Award（与微软） | web 设计行业唯一真正全球通用的"技术 + 设计"双料信用 |
| **Brand New** | 不是奖项是设计批评 | Bryony Gomez-Palacio + Armin Vit 运营；"Reviewed/Noted/Spotted/Linked" 四档 |

**判断**：客户是 B2B 大企业 → Red Dot / iF；web / 数字产品 → Awwwards；消费品牌想拿媒体头条 → Brand New / It's Nice That 报道。

## 七、设计教育商业化 3 模式

- **Domestika**：自营 + 录播，专业摄影棚自制课程，混合订阅+单课。**核心优势**：西语+英语双市场
- **Skillshare**：$9.95/月订阅（2021 砍免费层），累计 $108M 融资。**把"教设计"变成"网赚渠道"**
- **MasterClass**：年订阅制，估值 $2.75B（2021），用名人+电影级制作

**真正赚钱的独立设计课 4 共同点**：① 作者本人 50K+ 粉丝；② 课程有可交付物（不是"看完就好"，是"做完一个东西"）；③ 价格 $200-500，不是 $9.95；④ 终身可访问 + 私域社群。

## 八、AI 时代的设计商业新机会

传统设计服务边际成本是人小时，AI 重写了这个游戏：

1. **AI 工具评测与采购顾问**——帮中大型企业做"我们到底买哪个 AI 设计工具"决策，$5K-50K/项目
2. **AI 视觉资产定制**——Midjourney/Flux/Sora 做"AI 摄影/视频广告"，按传统摄影 30% 价格、10 倍产能
3. **Prompt 库与系统化 Prompt 工程**——按席位卖
4. **AI DesignOps**——帮企业把"AI 生成的混乱"接进 Figma/Linear/Notion 工作流，retainer $10K/月
5. **设计 token × AI 生成**——客户上传品牌 token，AI 按 token 自动生成合规变体
6. **AI 时代"反 AI 化"咨询**——专做去 AI slop 的高级定制，$30K+ 起
7. **AI 教育课**——教设计师用 AI 而不丢判断力，单课 $300-500
8. **AI 工作流播客/Newsletter**——把"AI 设计商业"做成细分媒体

## 7 条"卖设计的 5 种模式"

1. **可重复卖的产品**（课程、模板、字体、插件、token 库）> 一次性项目
2. **订阅化**（Retainer、SaaS、Newsletter、平台抽佣）> 单次收费
3. **价值定价** > 时薪定价
4. **平台抽佣 / Marketplace**（Domestika、Skillshare、Creative Market、UI8、Iconfinder）
5. **声誉套利**——先免费曝光建立权威，再高价接咨询
6. **混合栈**（产品 + 咨询 + 教育 + 订阅）
7. **企业内嵌**（DesignOps 顾问驻场）——客单价 $10K-50K/月

## 8 条项目定价反模式

1. **时薪报价当默认**——天花板低、跟客户利益对立（越慢越赚）
2. **不写修改轮数**——客户以为可以无限改
3. **不收 kill fee**——客户说"算了不做了"你白干三周
4. **不区分 discovery 和 execution**——策略+执行打包一个价
5. **接受"用结果付费"但没上限**——客户成功后嫌贵、失败时你背锅
6. **无 NDA / 无 IP 归属条款**——交付物所有权模糊
7. **单一货币报价**——汇率波动吃掉利润
8. **把"朋友价"和"市场价"混在一起**——一旦发现你会失去所有朋友

## 5 条"如果你只能记一条"

1. **设计商业化的胜负手是"可重复卖"**：把一次性项目产品化、把零散知识订阅化、把单次咨询模板化。Pentagram 50+ 年、Lippincott 9 办公室、Tailwind 养 50 人公司——都因为卖"系统"不是"工时"
2. **定价的本质是定价权**：永远不要主动报时薪——那等于告诉客户"我的时间就是我的上限"。先讲清楚业务价值、再说价格、最后谈条款
3. **AI 不会取代设计师，但会取代"不用 AI 的设计师"**：把 AI 当乘法器（× 产能），不当替代品。当你能用 AI 一天交付以前一周的工作，你的定价模型就该彻底重写
4. **先做作品 → 再做品牌 → 最后做平台**：Chimero 用了 10 年才写完一本书；Rendle 在 Apple 工作本身就是信用
5. **永远保留 20% 精力做"声誉资产"**：写作、开源、播客、奖项、演讲——这些不直接产生收入，但决定你下一年能接到什么样的客户
