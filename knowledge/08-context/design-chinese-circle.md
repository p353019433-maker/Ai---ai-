# 中文设计圈本土实践：执行手册

## 一句话总结
**中文设计不是"翻译成中文"，是"用中文的视觉语法重新设计"**。中文字符、标点、字号、行高、留白都和英文不同——英文 16px 好看，中文必须 14-15px；英文行高 1.5 好看，中文必须 1.7-2.0。**站酷、UI 中国、优设**是中文设计师的"GitHub"——学中文设计先看这些社区。

## 1. 中文设计 vs 英文设计 7 大差异

| 维度 | 英文 | 中文 | 差异原因 |
|---|---|---|---|
| **字号** | 14-16px | 14-15px | 中文字符密度高 |
| **行高** | 1.4-1.6 | 1.7-2.0 | 中文需要更多呼吸 |
| **字重** | Light 常见 | Medium 起步 | 中文 Light 难识别 |
| **字间距** | -0.01em | 0em | 中文字间距天然存在 |
| **标点** | 半角 | 全角 | 视觉重量不同 |
| **行宽** | 50-75 字符 | 25-40 字符 | 中文字符宽度 |
| **按钮高度** | 36-40px | 40-44px | 中文按钮更"挤" |

**核心**：中文排版的"呼吸"比英文多 30%。

## 2. 中文排版 5 大必处理

### (1) 字号偏大 1-2pt
```css
/* 英文 */
body { font-size: 16px; }

/* 中文 */
body { font-size: 15px; line-height: 1.8; }
/* 中文字号反而略小（因为字符密度高），行高反而大 */
```

### (2) 行高偏大
- 英文：1.4-1.6
- 中文：**1.7-2.0**（必须更大呼吸）

### (3) 标点挤压
- 标点占字宽（`：，。、；`）= 半字符宽度
- 行尾标点悬挂（Hanging Punctuation）
- 工具：CSS `hanging-punctuation: allow-end` 或手动调整

### (4) 避头尾
- 行首不能是 `、，。；：` 等标点
- 行尾不能是 `（` `"` `【` 等开口符号
- 工具：CSS `text-justify: inter-ideograph` 或排版引擎

### (5) 中英文混排
- 中文用中文标点（`，。！？「」（）`）
- 英文用英文标点（`, . ! ? "" ()`）
- 数字 / 英文单词前后空 1/4 字符

## 4. 中文 UI 设计 5 大原则

### 原则 1: 字符密度考虑
中文 1 个字符 = 英文 1.5-2 个字符宽度。同样 16px，**中文显示信息量比英文少 30%**。
- 解决：缩窄容器 + 减小字号（但保持行高）

### 原则 2: 字号下限 14px
中文 < 14px 难识别。**永远不用 < 14px**（移动端 < 12px 必崩）。

### 原则 3: 按钮更大
英文按钮 32-40px 高，中文 40-44px 高（因字符密度）。

### 原则 4: 标点全角
中文段落用全角标点（`，。！？`），英文段落用半角。
- 工具：自动转换库（如 `fullhalf`）

### 原则 5: 数字英文保留半角
- 数字（0-9）保留半角（`123.45`）
- 英文单词保留半角（`Stripe`）
- 中文段落中数字 / 英文前后空 1/4 字符

## 9. 中文设计 token 5 大特殊考虑

### (1) 字号 token
```js
// 英文 token
{
  fontSize: {
    xs: 12, sm: 14, base: 16, lg: 18, xl: 20
  }
}

// 中文 token
{
  fontSize: {
    xs: 12,    // 不可用
    sm: 14,    // 最小
    base: 15,  // 默认
    lg: 17,    // 略大
    xl: 20
  },
  lineHeight: {
    body: 1.8,    // 英文 1.5, 中文 1.8
    heading: 1.4
  }
}
```

### (2) 字体 token
```js
{
  fontFamily: {
    sans: ['"PingFang SC"', '"Hiragino Sans GB"', '"Microsoft YaHei"', 
           '"Source Han Sans CN"', 'system-ui', 'sans-serif'],
    display: ['"Source Han Serif SC"', '"Noto Serif CJK SC"', 'serif'],
    mono: ['"JetBrains Mono"', '"Cascadia Code"', 'monospace']
  }
}
```

### (3) 标点 token
```js
{
  punctuation: {
    full: '，。！？；：（）【】「」《》""…—',  // 中文标点
    half: ',.!?;:()[]<>""\'-',  // 英文标点
    prefer: 'full'  // 默认全角
  }
}
```

### (4) 字符间距 token
```js
{
  letterSpacing: {
    body: 0,           // 英文 -0.01em
    heading: 0.02em,   // 中文略大
    display: 0.05em    // 标题更大
  }
}
```

### (5) 行宽 token
```js
{
  maxWidth: {
    paragraph: '40ch',  // 中文 40 字符（vs 英文 66ch）
    article: '640px',   // 移动端
    desktop: '720px'    // 桌面
  }
}
```

## 10. 5 个中文排版 CSS 技巧

### 技巧 1: 标点悬挂
```css
.chinese-paragraph {
  hanging-punctuation: allow-end;
  text-align: justify;
  text-justify: inter-ideograph;
}
```

### 技巧 2: 中英文间距
```css
/* 自动给中英文之间加空格 */
.mixed-content {
  word-spacing: 0.05em;
}

/* 手动 */
.chinese::before,
.english::after {
  content: ' ';
}
```

### 技巧 3: 行高
```css
/* 英文 */
.en { line-height: 1.5; }

/* 中文 */
.zh { line-height: 1.8; }
```

### 技巧 4: 字号偏小
```css
/* 英文 */
.en { font-size: 16px; }

/* 中文（同样视觉大小）*/
.zh { font-size: 14px; }
```

### 技巧 5: 字间距
```css
.zh-heading {
  letter-spacing: 0.05em;  /* 中文标题略大 */
}
.zh-body {
  letter-spacing: 0;  /* 中文正文 0 */
}
```

## 14. 5 条"如果你在做中文设计，记住这些"

1. **中文排版 ≠ 英文排版翻译**——> 用中文视觉语法重设计
2. **行高 1.7-2.0**——> 英文 1.4-1.6 必崩
3. **字号 ≥ 14px**——> 永远不用 < 14px
4. **思源黑体 + 思源宋体**——> 免费 + 覆盖 95%
5. **中英之间空格**——> `中国 HTML5` 不是 `中国HTML5`

## 5 条反面教材

1. **翻译式设计**——> 中文段落英文标点 + 英文行高
2. **小字号**——> 12px 中文必崩
3. **小行高**——> 1.4 行高中文拥挤
4. **混排无间距**——> `中国HTML5` 难看
5. **忽略中文标点**——> 半角 vs 全角混乱

## 5 条"如果你只能记一条"

1. **中文排版 ≠ 英文排版翻译**——> 字号小 1pt + 行高大 0.3
2. **思源黑体 / 思源宋体**——> 免费 + 覆盖 95%
3. **行高 1.7-2.0**——> 永远比英文大
4. **字号 ≥ 14px**——> 永远不用 < 14px
5. **中英之间空格**——> 排版铁律
