# 日本与北欧设计哲学（循环1）

## Wabi-sabi 侘寂
- 接受"无常、不完美、未完成"
- 三大原则：nothing lasts / nothing is finished / nothing is perfect
- 设计特征：不对称、粗糙、朴素、谦逊、亲近自然
- 实例：ryōan-ji 枯山水、茶室粗陶碗、金缮、盆栽、ikebana
- 对设计的启发：不必追求完美与对称；接受材料本身的痕迹；让"使用过的质感"成为美

## Ma（间）— 留白的空间
- 不是"空"，是"关系"——元素之间的呼吸
- 来源：[尝试多个 URL 后未获取](https://en.wikipedia.org/wiki/Ma_(Japanese)) 404
- 核心思想：负空间本身是积极的设计元素
- 映射到 UI：留白=对内容的尊重

## Shibui 渋い — 静美七元素
1. Simplicity 简朴
2. Implicitness 含蓄
3. Modesty 谦逊
4. Naturalness 自然
5. Everydayness 日常
6. Imperfection 不完美
7. Silence 静默

**关键判断**：物体整体看似简单，但有微妙的细节（材质纹理）
**配色**：低饱和、混灰、产生"银"感
**与 wabi 区别**：shibui 不强调"刻意的不完美"，更克制、更理性
**Matthew May 总结**：Elegant simplicity. Effortless effectiveness. Understated excellence. Beautiful imperfection

## Muji 无印良品
- 哲学：Kanketsu（简素）— 在忙碌日常中带来安静的平静
- 流程：原型用手绘而非电脑，"以免鼓励不必要的细节"
- 反"为科技而科技"
- 品牌策略：完全无品牌、无广告、靠口碑

## Issey Miyake 三宅一生
- 设计应服务生活，不是抽象哲学
- 服装解放身体与情感
- 技术 + 工艺：先剪裁缝制，再用热压在纸层中产生永久褶皱
- 受 Vionnet 几何方法 + Brâncuși 雕塑启发
- "可穿可洗可旅行的雕塑"

## Hygge 丹麦/Norwegian
- 日常共处的温暖、平等、自发社交流动
- 10 个理想：atmosphere / presence / pleasure / equality / gratitude / comfort / togetherness / harmony / truce / shelter
- 对极简的影响：极简不冷，关键在材质 + 暖光 + 织物质感
- 反 ostentation，倡"understated elegance"

## Japandi
- 日本极简 + 北欧极简
- 配色：off-white / 米色 / 浅灰；自然光最大化
- 材质：石头 / 纸 / 竹 / 木
- 家具：低矮、木本色
- 光线：暖黄间接光
- 关键：东方"空"的克制 + 北方"暖"的舒适

## Scandinavian Design
- 三大原则：simplicity、minimalism、functionality
- 民主设计理想：让所有人都能享受好设计
- 关键设计师：Alvar Aalto / Arne Jacobsen / Hans Wegner / Verner Panton / Poul Henningsen / Aino Aalto / Kaj Franck / Tapio Wirkkala
- 材质特点：furniture, textiles, ceramics, lamps, glass
- 与 Art Nouveau 对立：反对装饰过剩
- 受 Bauhaus 影响但更人性、有机

## Kintsugi 金缮
- 修复不掩盖，痕迹被金色描绘
- 反对"计划性报废"
- 设计哲学：使用痕迹不是失败，是故事
- 框架：磨损、损坏、修复都是物体生命的一部分

## 对 AI 去 AI 化的具体指导
1. **不要"完美"**：AI 偏爱完美对称、加高斯模糊、磨皮——这与 wabi-sabi 背道而驰
2. **接受不完美**：留点材质纹理、留点非对齐、留点呼吸
3. **配色减色**：shibui 配色（低饱和 + 混灰）是 AI 蓝/紫/粉的反面
4. **用真材实料**：Muji 手绘原型 vs AI 即时出图
5. **Ma = 留白**：AI 倾向填满每一寸，应主动留白
6. **Kintsugi 思维**：让"做的痕迹"可见——手绘注释 > AI 干净图
