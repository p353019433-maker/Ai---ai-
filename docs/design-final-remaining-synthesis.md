## 5 个 Wikipedia 概念 + 3 个 Shades of X 真洞察

- **User interface (6,241 词)** —— X-sense 分类真有用：3-sense CUI = visual+audio+smell (Smell-O-Vision) / 4-sense VR/AR = 加 touch / "qualia interface" = 全感。**BCI/BMI 有趣：no input device other than electrodes = UI = 神经系统本身**
- **Flexbox (1,519 词)** —— 历史：2009-07-23 起 / 2018-11-09 Candidate Recommendation / 2021-03-25 Working Draft preview。**基于 Mozilla XUL (Firefox)**。2022-12 99.68% 浏览器支持。Editor lineage: Tab Atkins Jr. / Elika J. Etemad / Rossen Atanassov (current); Ian Hickson (former, also HTML5 editor) = tell
- **Mockup (2,099 词)** —— **2 个真洞察**：
  1. **服装 "toiles/muslins" 是"故意部分未完成"** (一只袖子、一只尾巴、raw seams) = 迭代调整 cheap 化 = **不是 shortcut 是 strategy**
  2. "You can fix it now on the drafting board with an eraser or you can fix it later on the construction site with a sledge hammer"
- **Building design / Numero sign** —— Wikipedia stubs, 无设计洞察
- **Shades of orange/gray/green (3 篇)** —— **真设计价值**：
  - **Web "orange" `#FFA500` (hue 39°) ≠ color-wheel orange `#FF8000` (hue 30°)**。CSS `color: orange` 给的是 yellow-shifted 不是纯
  - Safety orange `#FF7900` (ANSI Z535); International Orange (aerospace) `#FF4F00`; (Golden Gate Bridge) `#F04A00`
  - **CSS 所有 "chromatic" grays 都是 cool grays** (slategray `#708090` / light `#778899` / dark `#2F4F4F`)，**没有 built-in warm gray** = skin-tone-friendly neutral 要手挑 slightly-red-shifted hex
  - **sRGB / HTML/CSS name clash for green**: sRGB "green" `#00FF00` = HTML/CSS `lime`; HTML/CSS "green" = `#008000` (Office Green)。CMYK 绿 `#00A550`。`color: green` 在 web 渲 `#008000` 不是 `#00FF00` = **学 RGB 原色图的设计师会踩坑**
  - **结论**: **CSS named colors ≠ 学校学的 colors**。safety/aerospace/national-identity 颜色有自己的 standardized hexes **不是 CSS named 的**

## 7 条具体设计决策/洞察
1. **YHSong 跨域方法 catalog 是 working design system** —— pair 스타카토 + 체리 피킹 + 망치와 모루 전술 在一个 numbered index = **method = transferable primitive**。设计团队可以列 30 跨域方法当 critique rubric
2. **Dinamo Gravity = editorial display 一站式** —— 1 文件 + 16 SS = 不需买第二个 display face
3. **KOMUNIQUE 配色 = 故意非-Swiss 选** —— 酸绿+深紫 = **小城瑞士工作室 mobile-first 设计**
4. **DOSSIER 透明定价 = 独立出版 template** —— 4-7 titles / cycle, sold-out = success, small editions, content-derived form
5. **Henning Larsen 评奖标准 = 名字骗人** —— "European" 实际评 global influence
6. **Pakistan 旗 = 41 年 retrofitted 国族图标** —— 1906 设计 → 1947 国家 → 1985 PSQCA 重标准化 → 7 色白条承认 minorities
7. **CSS named colors ≠ 学校 colors** —— `color: orange` = yellow-shifted not pure; `color: green` = `#008000` not `#00FF00`; all chromatic grays are cool = **没有 built-in warm gray**
