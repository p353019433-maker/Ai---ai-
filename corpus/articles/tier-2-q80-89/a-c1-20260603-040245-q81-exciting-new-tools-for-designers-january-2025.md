It’s the first Monday of a new year, and many of you are back at work after a much needed break. Let us ease you back in with our pick of the latest tools to help you get stuff done faster, smarter, and with a little less stress. Enjoy! Comet UI Comet UI offers a curated set of interactive, accessible components to supercharge development speed. Built with Headless UI, Tailwind CSS, React, and Framer Motion, these customizable components are ready to use straight out of the box. Rapid 301 Rapid 301 can help you generate 301-redirects in a fraction of the time. Input URLs or upload sitemaps to receive .csv and .htaccess files in seconds. Review, implement, and test to maintain seamless link equity. Yenbek Yenbek task planning app organizes your tasks on a map, and presents you with one task a time. It should help you stay focused, complete each task fully, and reduce unnecessary stress. Layer Amid a sea of largely similar productivity tools, Layer takes a different approach. It organizes tasks visually and intuitively, in alignment with natural human thought processes. Mac Pet It can get a little lonely sometimes as a freelancer so how about a little canine or feline companion that lives in your menu bar? It wakes as you work and snoozes when you take a break. UI Inspector UI Inspector is a Chrome extension that lets you style and edit web pages in real-time. Streamline design and debugging by visually inspecting and editing elements directly on the page, bypassing DevTools to make quick, on-page adjustments with ease. Font Tester Font Tester is a browser extension designed to help you select the best font for a project by letting you preview over 1,000 Google Fonts directly on your website. Highlight some text, select a font to view, and when you’ve decided, copy the code to load that font. Minimox Minimox is a simple to use minimalist logo generator web app. Choose from a range of suitable typefaces, icons and colour schemes, adjust text and image positions, and download the finished logo as SVG or PNG. Great for placeholders or mockups. DesignWhiz V2 DesignWhiz 2 is a new platform for designers to share work, collaborate, and look for work. It’s still early days, but there is already a wealth of resources available, and membership is free. Toaster Discover how users engage with your Figma prototype by creating and sharing a unique tracking link with Toaster. You can track record user session info like location and duration, and view how individual users interact with your prototype. RightFont 9 RightFont is a versatile font management tool for Mac that simplifies organizing, previewing, and activating fonts. It streamlines font selection, improves workflow, and supports collaboration for creatives and teams. Version 9.3 is the latest update, with improved customisation. PureAnim Use PureAnim to create SVG CSS keyframe animations visually and without code. Animations can be exported in various formats including CSS, JSON, and GSAP. Pie Menu Pie Menu opens a customizable menu of your most used shortcuts in a variety of Mac apps, including Illustrator, Photoshop, Affinity Photo and Designer, Sketch, Todoist, Slack and many more. Instead of remembering the shortcuts for the 6 most often used commands for each app, you only need to remember one. UI/UX Essential Icons Featuring 30 modern and adaptable icons, this set is ideal for app developers, web designers, and creatives seeking to build intuitive digital experiences. Covering navigation and interactions, it combines functionality with visual appeal. VSCode Pokemon If you use Visual Studio Code, you can liven up your coding experience by adding some Pokemon to your editor. These little guys don’t do anything as such, but they may just make you smile. Paddi MacDonnell Paddi MacDonnell is a designer and entrepreneur from Northern Ireland, follow her on Twitter.
