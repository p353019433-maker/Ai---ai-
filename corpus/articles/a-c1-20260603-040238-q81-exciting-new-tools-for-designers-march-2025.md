This roundup is packed with new goodies that can help you with daily tasks and more long-term projects. Enjoy! Beatoven Projects that need background music can be a challenge when it comes to finding just the right beat. Beatoven lets you create your own music using artificial intelligence and further customize those tracks to fit your projects perfectly. Start with a text prompt, and you will be on your way to creating music. Genaimo Another AI tool to help with complex tasks, Genaimo enables you to create simple animations. Start with a prompt and turn it into four motion elements that can help push your creative idea forward. Once you get it right, you can add more customizations and export your new moving graphic. Responsive Typographic Scale Generator Plan perfect type sizes for every web project every time with DK Web Solutions Responsive Typographic Scale Generator. The tool helps you create a strong visual hierarchy and responsive font sizes. Select a type scale of your choice and grab the responsive CSS code for your website. Rainbow Design Marrying design and code, Rainbow Design is a modern code editor that makes HTML, CSS, and JS designable. Write code on the screen, and you can see those changes in the design when the code changes. (This goodie will make front-end designers and developers happy, for sure!) Plus, the interface is super simple and easy to understand, which can be great for designers experimenting with some coding. Coolicons (Figma) Everyone needs a great icon set, and Coolicons includes more than 400 simply designed elements that work right in your Figma project. Plus, every icon works with Figma’s native component overrides for stroke and color changes so everything matches your design style guide. MoreDraw Boost collaboration and creative thinking with an online whiteboarding workspace. Create and connect nodes for text-based mind maps, flowcharts, notes, to-do lists, and icons. Perfect for brainstorming, planning, and organizing tasks. The canvas has no borders and works with a variety of file types. FeedbackStream When it comes to understanding users and their journeys, a strong feedback tool can help you gain the understanding you need. FeedbackStream automates customer interviews with AI so you can get the answers you need to improve your product. More in-depth than a survey, yet less time-consuming than 1:1 interviews. Envato Figma Plugin Most website designers are familiar with Envato, which includes a huge repository of tools from themes to icons to stock images and videos. Now, that tool can be integrated with your Figma workflow using a plugin. It even includes an icon generator! Blaze IDE: Python Code Editor Blaze is a powerful Python IDE and compiler for Android, allowing you to write, run, and debug Python code on the go. Whether you’re a Python beginner, student, or professional developer, Blaze provides an intuitive mobile coding experience with offline support, module installation, and GitHub integration. Yapz One more AI model for you to try with projects is Yapz. This tool turns questions and forms into conversational elements that you can use to create viral content. Story Magician Design something just for the kids with Story Magician. This AI tool mixes writing and design to help you create a custom story with images. Just describe your story idea – be as imaginative as you can – and bring it to life with images. You can upload photos so the characters resemble your kids, too! When you are ready, generate a story with illustrations and make adjustments to create something fun and completely customized based on your idea. Khroma Create super immersive color palettes that are rooted in the colors you like best with Khroma. The tool uses AI to learn which colors you like – start by picking 50 hues – and then generate dozens of color pairs that look great. This is a good brainstorming tool when you are stuck with branding projects or thinking about color palettes. Shogun For e-commerce projects, there’s always a balance between function and great design. Shogun is a landing page builder tool that works with your Shopify site. It is a visual drag and drop page builder that can help you better customize and bring your vision to life on your Shopify site quickly. And if you aren’t sure where to start, they have landing page templates to jumpstart your creative thinking. Cloner & Repeater (Figma) Don’t keep drawing the same components over and over again when you can clone and repeat. The Cloner & Repeater Figma plugin is designed to clone any single selection or group and use it as many times as you need. Swipeart Tired of not finding art that speaks to you during museum visits? This app helps you discover breathtaking artworks with a simple swipe, refine your artistic taste, and preview museum collections before you visit. (It’s also a great source of creative inspiration when you are stuck on a project.)
